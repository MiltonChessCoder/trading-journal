# 📈 Trading Journal App

A fullstack trading journal application that allows users to track, analyze, and visualize their trades.

---

## 🚀 Features

* Add, edit, and delete trades
* Automatic profit/loss calculation
* Dashboard with key metrics (total profit, win rate, total trades)
* Performance chart (trade-by-trade growth)
* Clean fullstack architecture

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Recharts (data visualization)

### Backend

* Node.js
* Express

### Database

* MongoDB (via Mongoose)

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/MiltonChessCoder/trading-journal.git
cd trading-journal
```

---

### 2. Setup Backend

```
cd backend
npm install
```

Create a `.env` file in the backend folder:

MONGO_URI=your_mongo_connection_string
PORT=5000
```

Run backend:

```
node server.js
```

---

### 3. Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🌐 Usage

* Open: http://localhost:5173
